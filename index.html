Silver Age Minecraft Server

Description

We run an official server for r/SilverAgeMinecraft, running on 1.5.2 with minimal QoL plugins for player experience.
In addition to a traditional survival world, we have incorporated a classic "Beta" terrain style world as well as a skylands world (like the Aether!). As well as utilising the mobarena plugin to provide some RPG/Dungeon style gameplay and extra rewards.
